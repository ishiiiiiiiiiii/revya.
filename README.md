# RevyaApp

A simple fitness and health tracking app built with:

- **React Native (Expo)** — for the mobile frontend
- **Express + Node.js + MongoDB** — for the backend API
  How to Run
  Backend:

# Install dependencies

npm install

# Start server

node server.js

Frontend (React Native with Expo):

# Install dependencies

npm install

# Start Expo

npx expo start
